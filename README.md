# PDF安全工具 - Windows 11 免安装版

一个免安装的EXE工具，用于在Windows 11上对PDF文件进行水印添加和加密。

## 📦 功能特点

- ✅ **免安装EXE** - 直接运行，无需Python环境
- ✅ **图形界面** - 简单易用的GUI操作
- ✅ **添加水印** - 自定义文本、透明度、角度、颜色
- ✅ **PDF加密** - 设置用户密码和权限控制
- ✅ **批量处理** - 支持文件夹批量操作
- ✅ **中文界面** - 完全中文操作界面

## 🚀 快速开始

### 方法1：直接使用预编译EXE
1. 下载 `PDF安全工具.exe`
2. 双击运行即可
3. 无需安装任何依赖

### 方法2：从源代码打包（如果需要自定义）
```bash
# 1. 安装Python 3.8+
# 2. 安装依赖
pip install PyPDF2 reportlab pyinstaller

# 3. 打包EXE
python build_exe.bat
```

## 🎯 使用方法

### 1. 添加水印
1. 打开工具，选择"添加水印"标签页
2. 选择输入PDF文件
3. 设置水印文本和参数：
   - 文本：如"机密文件"、"草稿"等
   - 透明度：0.1-0.8，数值越小越透明
   - 角度：水印旋转角度
   - 字体大小：20-80
   - 颜色：灰色、红色、蓝色等
4. 指定输出文件路径
5. 点击"开始添加水印"

### 2. PDF加密
1. 选择"PDF加密"标签页
2. 选择输入PDF文件
3. 设置密码：
   - 用户密码：打开PDF需要的密码
   - 所有者密码：高级权限密码（可选）
4. 设置权限：
   - 允许打印：是否可以打印
   - 允许复制：是否可以复制内容
   - 允许修改：是否可以修改文档
   - 允许注释：是否可以添加注释
5. 指定输出文件路径
6. 点击"开始加密"

### 3. 批量处理
1. 选择"批量处理"标签页
2. 选择输入目录（包含多个PDF文件）
3. 选择输出目录
4. 选择操作类型：
   - 添加水印
   - PDF加密
   - 同时添加水印和加密
5. 设置相应参数
6. 点击"开始批量处理"

## 🔧 高级选项

### 命令行版本
如果需要命令行版本，可以使用 `pdf_secure_tool.py`：

```bash
# 添加水印
python pdf_secure_tool.py watermark input.pdf output.pdf "机密文件"

# 加密PDF
python pdf_secure_tool.py encrypt input.pdf output.pdf mypassword

# 批量处理
python pdf_secure_tool.py batch ./input ./output watermark "公司机密"
```

### 自定义打包
修改 `pdf_secure_gui.spec` 文件可以自定义打包选项：
- 更改程序名称
- 添加/删除依赖
- 调整图标
- 设置版本信息

## ⚠️ 注意事项

1. **备份文件**：处理前请备份原始PDF文件
2. **密码安全**：加密后请妥善保管密码，丢失无法恢复
3. **文件权限**：确保有足够的文件读写权限
4. **测试验证**：批量处理前建议先测试单个文件
5. **系统兼容**：建议在Windows 11上使用，兼容Windows 7+

## 🐛 故障排除

### 问题1：EXE无法运行
- 确保系统是64位Windows
- 尝试以管理员身份运行
- 检查杀毒软件是否误报

### 问题2：处理失败
- 检查PDF文件是否损坏
- 确保文件不是只读状态
- 检查磁盘空间是否充足

### 问题3：界面显示异常
- 尝试调整系统显示缩放设置
- 确保系统语言设置为中文
